Unterschied Server/Arbeitsplatz Lizenz

Die Server Lizenz ist technisch identisch zur Arbeitsplatz Lizenz (und zur 30 Tage Trial-Version), der Unterschied besteht ausschließlich im Preis.
Wenn auf der Maschine, die Sie sichern, Windows XX Server installiert ist, benötigen Sie eine Serverlizenz.
Ansonsten genügt eine Arbeitsplatzlizenz. Für jede Maschine, die gesichert werden soll, benötigen Sie eine eigene Lizenz.
Die Lizenz beinhaltet zwei Jahre freie Updates.
