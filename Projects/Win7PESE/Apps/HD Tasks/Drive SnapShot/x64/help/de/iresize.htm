Snapshot - Partition Resize

Partitionen vergrößern/verkleinern

Es wird manchmal gewünscht, Partitionen zu vergrößern oder zu verkleinern.

Drive Snapshot unterstützt dies ab 30.September 03 unter folgenden Voraussetzungen:

  • Es wird ausschließlich das Filesystem innerhalb der Partition verändert, die Partition beibt (derzeit) unverändert (wir arbeiten dran)
  • Vergrößern/Verkleinern ist nur unter Windows möglich
  • derzeit wird ausschließlich NTFS unterstützt
  • das Resize ist ausdrücklich 'BETA'; d.h.
    der Code ist getestet, und funktioniert auch, ist aber weit weniger gut getestet als die übrigen Komponenten von Drive Snapshot.
    Sie verwenden Ihn auf eigene Verantwortung.
  • Bitte beendigen Sie alle Anwendungen, die auf das zu ändernde Laufwerk zugreifen, soweit das möglich ist.

Resize ist derzeit ausschließlich von der Windows Kommandozeile verfügbar.

Ausgabe der Resize Möglichkeiten (keine Änderung) mit 

            snapshot -!resize c:

min.
possible size: kleiner als diese Größe kann das Laufwerk nicht werden da an dieser Stelle Daten liegen; evtl.
kann dies durch einen Defragmentierungslauf verkleinert werden.
Es ist aber in der Regel nicht möglich, ein Laufwerk kleiner zu machen als ca.
0.5* die beim Anlegen des Laufwerks angegebene Größe, da an dieser Stelle einige Verwaltungsdaten liegen.
Insofern kann ein 10 GB Laufwerk nie mehr kleiner als 5GB werden.

Danach Änderung, in diesem Fall soll das Laufwerk von 2400 MB auf 3000 vergrößert werden.
Dabei findet folgendes statt:

Zunächst wird das Laufwerk auf Fehler geprüft; fehlerhafte Laufwerke rührt Snapshot nicht an.
In diesem Fall müssen Sie von Hand und von der Kommandozeile CHKDSK X: /F aufrufen, die Überprüfung per Windows (KlickKlick) findet einige Fehler nicht!

Danach wird das Laufwerk verkleinert, und in einen kontrolliert fehlerhaften Zustand gebracht, den wiederum CHKDSK beseitigen soll und kann und macht.

Sollte das zu ändernde Laufwerk Ihr Systemlaufwerk sein, oder sonst von einer Applikation belegt sein, fragt CHKDSK nach, ob er trotzdem laufen soll, bzw.
ob diese Prüfung beim nächsten Neustart vorgenommen werden soll.

Beantworten Sie alle Fragen mit Ja, und benutzen Sie das Laufwerk nicht, bevor CHKDSK nicht fehlerfrei durchgelaufen ist; wenn nötig starten Sie Windows sobald wie irgend möglich neu!!
