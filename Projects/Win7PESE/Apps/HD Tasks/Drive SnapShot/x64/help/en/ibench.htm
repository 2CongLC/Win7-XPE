Disk Backup performance data

Here are some numbers on Snapshot disk imaging performance, comparing both speed and compression ratios for certain configurations to other disk imaging software.

Please note however:
  • All these data are data and configuration dependent.
  • The measurements were performed by ourselves, so you shouldn't rely on there objectivity.
    However no attempt was done to produce fake data.

    Snapshot vs.
    PQDI vs.
    GHOST

    The comparisons were done using PQDI 3.0 and Ghost 5.0.
    For both products, newer versions exist.
    Should these perform significantly better, we apologize.

    Configurations tested:
  • P200MMX, 64MB memory,ATA66 DOS 6.2, HIMEM+SMARTDRV
  • P200MMX, 64MB memory,ATA66 WinNT4.0
  • PIII700, 256MB memory,ATA100 DOS 6.2, HIMEM+SMARTDRV
  • PIII700, 256MB memory,ATA100 Win2K
  • Dual PIII700, 256MB memory, Win2K

    The tested hard drive was always the same: 30 GB IBM, 5400 rpm.
    The copied partition was in all cases the P200 boot drive with WinNT4.0,FAT16, size = 1 GB, 864 MB used.
