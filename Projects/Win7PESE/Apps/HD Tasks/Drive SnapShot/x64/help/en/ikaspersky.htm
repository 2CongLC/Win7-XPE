Drive Snapshot - Kaspersky Antivirus

when running Drive Snapshot, you get an error similar to

        can't open \\.\C: : Windows error 5 - access denied

you have to tell Kaspersky Internet Security that we belong to the good guys as follows

Kaspersky Internet Security 2010

according to a customer (Kaspersky Internet Security 2010 is no longer available for download)

    "My Security Zone" ->"Application Activity" ->"category" change to "restricted"
    And then continue as shown below
 

Kaspersky Internet Security 2011

Protection Center -> Files and Data Protection -> Application Control -> Settings

Applications

move Drive Snapshot into the 'Trusted' group

click a couple of times [OK]
