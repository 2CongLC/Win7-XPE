Schritt 1 - wir brauchen etwas das bootbar ist

Wenn Sie ein Floppy Laufwerk haben:
Erzeugen Sie eine bootbare Floppy, am einfachsten durch 
   Start->Programme->Snapshot->Create Recover Floppy
verwenden Sie A: als bootbares Medium unten

Wenn Sie kein Floppy Laufwerk haben - Methode A:
Legen Sie eine kleine Partition an, und restaurieren Sie networkbootdisk.sna (installiert während Setup) in diese Partition
Diese kann jetzt evtl.
von Ihnen noch angepasst werden.
Verwenden Sie diese Partition als bootbares Medium unten.

Wenn Sie kein Floppy Laufwerk haben - Methode B:
erzeugen Sie ein bootbares (dekomprimiertes) Image einer Floppy
von der Windows Kommandozeile expandieren Sie networkbootdiskette.sna mit dem undokumentierten Kommando

       c:\programme\snapshot> snapshot networkbootdiskette.sna bootdisk.img -X

Verwenden Sie bootdisk.img als bootbares Medium unten.

Schritt 2 - brennen einer bootbaren CD/DVD mit NERO

  • starten Sie Nero
  • Legen Sie ein neues Projekt an
  • Schließen Sie den Wizzard, wenn nötig - wir brauchen den 'Experten'-Modus
  • links oben, selektieren Sie CD oder DVD
  • links unten suchen Sie CD-ROM [boot] oder DVD-ROM [boot]
  • klicken Sie den [boot] Tabulator
  • selektieren Sie (*) bootbares Laufwerk oder (*) Image File, je nach Medium oben
  • selektieren Sie Experten Ansicht
  • stellen Sie sicher, dass 'Emulation' auf Floppy Emulation 1,44 MB steht
  • [NEU]
  • jetzt können Sie der CD weitere Dateien hinzufügen; diese sind nach Boot unter W: erreichbar
  • brennen
  • fertig

STEP 3 - neu starten und verifizieren dass die CD booted

besser jetzt als erst im Notfall ;)
