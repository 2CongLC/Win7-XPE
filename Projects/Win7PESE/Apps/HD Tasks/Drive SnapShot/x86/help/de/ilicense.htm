Lizenz Informationen eingeben

Wenn Sie SNAPSHOT gekauft haben, erscheint dieser Dialog nicht.
Diesen Dialog brauchen Sie nur, wenn Sie aus einer heruntergeladenen Testversion eine vollwertige Version machen wollen.
Ihre Lizenzinformationen finden Sie auf unserer Rechung und auf dem Liefermedium (\license.txt).

Ohne gültige Lizenz Informationen ist die Nutzung von Snapshot zu Testzwecken auf 30 Tage beschränkt, danach ist nur noch ein Restore möglich.

Um SNAPSHOT danach weiter zur Datensicherung nutzen zu können, müssen Sie SNAPSHOT kaufen.

Diese zeitlich limitierte Nutzung gibt Ihnen die Möglichkeit, Snapshot in aller Ruhe zu testen (und auch mit Konkurrenzprodukten zu vergleichen), bevor Sie sich (hoffentlich) zum Kauf entschließen.

Wenn Sie Snapshot kaufen, erhalten Sie eine 4-zeilige Lizenz Information.
Diese besteht aus ihrem Firmennamen - UltraMegaSoft
eine weitere Zeile (für ihre Abteilung) - software Development
einer Seriennummer und Angabe über ihre Lizenz (1Workstation, 2 Server, Firmenlizenz) 
und einer 'Unterschrift' unsererseits, die die Gültigkeit obiger Lizenzinformation bestätigt.

Tragen Sie diese bitte oben in das weiße Feld ein, und klicken [Register].

Daraufhin wird SNAPSHOT.EXE personalisiert; d.h ihre Lizenz Informationen werden Bestandteil der EXE-Datei.
