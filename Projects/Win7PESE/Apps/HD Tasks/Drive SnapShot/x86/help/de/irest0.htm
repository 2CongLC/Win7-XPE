Drive Snapshot kann ein Laufwerk vollständig restaurieren.

Das Vorgehen wie folgt.

Zunächst entscheiden Sie, ob Sie eine gesamte Festplatte mit mehreren Partitionen, oder eine einzelen Partition restaurieren möchten.

Wenn Sie eine gesamte Festplatte restaurieren möchten, wählen Sie "Restore complete disk from images" Achtung! Das Restaurieren der gesamten Festplatte mit dieser Methode funktioniert nur dann, wenn alle Partitionen gleichzeitig mit einer Snapshot Version ≥ 1.45 gesichert wurden. 

 

Wenn Sie nur eine einzelne Partition restaurieren möchten, wählen Sie "Restore disk from file"
