Snapshot - DiskImage Restore for Windows NT/2000/XP

Test the Snapshot Userinterface
Obviously, a disk image program would be of little value without a restore.
Snapshot does restore disk images, but has some limitations.
However, these are the same limitations that apply to FORMAT and CHKDSK, which result from the fact that WinNT won't allow access to open files.
So most important, the system drive and all drives with a PAGEFILE cannot be restored while WinNT is running.

To restore a System Volume you have to use  DOS restore.

All other volumes can be restored from WinNT.
