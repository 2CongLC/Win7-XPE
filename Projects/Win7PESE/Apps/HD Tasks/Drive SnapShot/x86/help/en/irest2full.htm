Screenshots

At the top of the window you have to select one of the images saved with Snapshot. The associated images, saved at the same time, will be found automatically.

In the top half of the window you see a list of all images saved at the same time. All images are selected for restore by default.

Below you have to select the destination disk. The graphics at the bottom of the window shows the current partitioning of the destination disk.

Warning: All data on the destination disk will be overwritten..
If you need any files from the destination disk you have to make a backup before continuing!

If all parameters are selected press [Next] to continue.
